# terraform
블로그를 통해 AWS 인프라를 Terraform 이용하여, 구성하고 소스 코드를 정리할 예정입니다.

# Security Group

1. default_sg.tf
기본 리소스 생성 코드를 이용한 security group(보안 그룹) 생성 코드

2. module_sg.tf
모듈 코드를 이용한 security group(보안 그룹) 생성 코드
